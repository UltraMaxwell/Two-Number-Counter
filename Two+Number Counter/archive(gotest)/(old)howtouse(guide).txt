linux: (spoiler: there is no windows and macos rn)

1. DOWNLOAD AND INSTALL GO:
    
    1.1. download go from official Go website (go.dev/dl) or with wget:
    
    wget https://go.dev/dl/go1.25.4.linux-amd64.tar.gz
    
    1.2. uninstall the old version of Go (if you have one) and install the new one:
    
    sudo rm -rf /usr/local/go && sudo tar -C /usr/local -xzf go1.25.4.linux-amd64.tar.gz
    
    1.3. add the path to Go to your profile file (~/.bashrc or ~/.zshrc):
    
    echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
    source ~/.bashrc
    
    1.4. check if Go is installed:
    
    go version (if the terminal displays "go version go1.25.4 linux/amd64" or something like that, then Go is installed)
    
2. HOWTOUSE "Two+Number Counter":
    
    2.1. open “Two+Number Counter” folder in terminal
    
    2.2. type "go run main.go"
    
    2.3. enter the two numbers you need to add (IMPORTANT: the program will not display a number containing more than 9 digits)
    
    2.4. click "enter" / "return"
